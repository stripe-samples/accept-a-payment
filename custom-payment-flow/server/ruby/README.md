# Accept a payment

A [Sinatra](http://sinatrarb.com/) implementation.

## Requirements

- Ruby v2.4.5+
- [Configured .env file](../../README.md)

## How to run

1. Install required dependencies

```
bundle install
```

2. Run the server locally

```
ruby server.rb
```

3. Go to [http://localhost:4242](http://localhost:4242) in your browser
